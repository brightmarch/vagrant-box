A Debian Jessie 8.10 64-bit box that includes:

* Compass 0.12.4
* Go 1.4.2
* Node 0.12.7
* PHP 5.6.11
* php-imagick 3.1.2
* php-redis 2.2.7
* php-ssh2 0.12
* php-zmq 1.1.2
* Postgres 9.4.4
* RabbitMQ 3.5.3
* Redis 3.0.2
* Ruby 1.9.4
* SASS 3.3.4
* ZeroMQ 4.0.4
* ZURB Foundation 4.3.2
